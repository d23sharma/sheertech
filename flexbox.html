Flexbox


Flex-box is a one-dimenional layout method for laying out items or sharing space or distributing space or adding space in either rows or columns to make a webpage.
Flexbox is a better way to align items within a container.
A Flexvox is 2 axis which are main axis and cross axis.
To activate flexbox or to use the flexbox layout we use the property display: flex
A Flexbox container will always have a direction either row (horizontal) or column (vertical) direction, basis the direction it will have main axis and cross axis
If we select flex-direction: row then row will become the main axis or if we select flex-direction: column then column will become the main axis.
By default main axis is row (horizontal) which goes from left to right.
Flex properties can only be applied to the direct children.
Flex box is a block level element.
Flexbox helps us to distribute space dynamically across elements of an unknown size, hence the term flex.

CSS FLEXBOX

The main concept of flexbox is to how do we distribute space across different elements to make those elements larger or just add space in between these elements to move them to the left, to the right, split up the space evenly, make sure that the first thing gets more space or the middle thing gets the least amount of space with in one container.
We can create flexible layouts. The boxes, the elements inside are flexing or flexible.

Important :

To make a layout Flex there are certain pre-requisites:

  • Flexbox properties can only be applied to its direct children
  • The parent element will always be a container
  • To give a layout flexbox attributes we need to use the property display: flex
  • By default flex-direction is row which means it goes from left to right, to make the flex direction to be column which goes from top to bottom, we need to assign the property flex-direction: column
  • flex-direction determines how the content flows in a container
  • The flex-direction we assign becomes the main axis and the other direction will be cross axis
  • Inside of the container box comes the flex-item

The properties associated with container are :
  1. flex-direction
  2. justify-content
  3. flex-wrap
  4. align-item
  5. align-content
The properties associated with flex-item are :
  1. order
  2. flex
  3. flex-grow
  4. flex-shrink
  5. align-self
Container
1. flex-direction :     It helps to determine how the content flow whether its left to right, right to left, top to bottom or bottom to top.
# Property Declaration
1 flex-direction: row; To assign the flexbox direction from left to right in a row
2 flex-direction: row-reverse; To assign a flexbox direction from right to left in a row
3 flex-direction: column; To assign the flexbox direction from top to bottom
4 flex-direction: column-reverse; To assign the flexbox direction from bottom to top
2. flex-wrap :     Specifies whether our elements are going to wrap along the main axis onto a new line if it's horizontal or a new column if it's vertical main axis, in other words it specifies whether flex-items are forced into a single line OR can be wrapped into multiple lines.
# Property Declaration
1 flex-wrap: nowrap; It squeezes the items in the same row or column also the default state for an item is no wrap
2 flex-wrap: wrap; it does not squeeze the content with in the same row or column and it allows to move the content to the next line
3 flex-wrap: wrap-reverse; Specifies that the flexible items will wrap, if necessary, in reverse order. If there is any element to reverse so it wraps in reverse direction in cross axis.
NOTE :     Flex-direction and Flex-wrap are the two properties we use to control the 2 axis which are main axis and cross axis. 3. Justify-content :    Defines how space is distributed between items in flex container along the main axis or how the actual element, the content is distributed across the main axis depending upon the flex-direction
# Property Declaration
1 justify-content: flex-start; Default value. Items are positioned at the beginning of the container
2 justify-content: flex-end; Items are positioned at the end of the container
3 justify-content: center; Items are positioned in the center of the container along the main axis
4 space-between; Items will have space between them except for the first and the last element as it does not leave any space on the outside edges.
5 justify-content: space-around; Items will have space before, between, and after them like equal margins
6 justify-content: space-evenly; Items will have equal space around them
4. align-items     Defines how space is distributed between items in flex container along the cross axis. It is very similar to justify contents but justify contents is for the main axis and align-items is for cross axis.
# Property Declaration
1 align-items: flex-start; It aligns the item to the start of the column.
2 align-items: flex-end; It aligns the item to the end of the column.
3 align-items: center; It aligns the item to the center of the column along the cross axis.
4 align-items: stretch; The default property for align items is stretch as it stretches to fit the cross axis.
5 align-items: baseline This property helps to align the text content with in the element
5. Align-content:     It defines how space is distributed BETWEEN ROWS in flex container along the cross axis. It modifies the behaviour of flex-wrap property. It does nothing if we don't have flex-wrap property assigned to an element. It's a mixture of justify contents and align items. It distributes additional space along the cross axis. It is very important to have multiple element e.g multiple rows or colums to make align content work.
# Property Declaration
1 align-content: stretch; Its a default property, it is used when the elements are over-flowing the container and we apply flex-wrap property so the element which were overflowing will come in the next line.
2 align-content: flex-start; It positions the elements at the top left corner if the flex-direction is set to row.
3 align-content: flex-end; It positions the elements at the bottom left corner if the flex-direction is set to row.
4 align-content: center; It positions an element in the center on the cross axis.
5 align-content: space-between; It position the elements at the top and at the bottom of the container and provide all the space in the middle.
6 align-content: space-around; It assigns equal margin at the top and at the bottom on cross axis
7 align-content: space-evenly; It assigns the equal space at the top and at the bottom of each of the elements on cross axis
Important: Align-item is used in single row but align-content is used in double row where the element size are bigger and not fitting with in the same row after applying flex-wrap: wrap property. It is used to adjust the space in between the rows after wrapping the elements. These are the properties we apply to the individual items that are inside of a flex container, using this property we can manipulate one item at a time.
Flex-items 1. align-self:     This is the first property which doesnt directly gets applied to the flex container instead it gets applied to the flex-items inside of flex container, It is very similar to align-items except it's actaully a property we add to a single element or it doesn't have to be one it could be more than one in a flex container. It helps us to change alignment along the cross axis for a single element using it.
# Property Declaration
1 align-self: center It centers the targeted element in the center of the container
2 align-self: flex-start It move the targeted elements to the top of the container if align-content is set to flex-bottom
3 align-self: flex-end It moves the targeted element to the bottom of the container is the align-content is set to flex-start.
4 align-self: stretch The targeted element is positioned to fit the container
5 align-items:baseline The element is positioned at teh baseline of the container
Important: The align-self property overrides the flexible container's align-items property.

Flex sizing property

Flex sizing property consist of 3 separate properties flex-basis, flex-grow and flex-shrink, they are meant for the individual items in a flex container, it specifies how the element should grow or shrink when there is available space or when there is too little space.

2. Flex-basis:    It specifies the initial length or size of a flex item BEFORE it's placed into a flex container
flex-basis' values could be in px or %.
It's basically used to handle width of flex-items.
# Property Declaration
1 flex-basis: 200px It specifies the value in pixel for the element which will have a different lenght
2 flex-basis: 20% We can assign the value in percentage as well to define the length of the specific element
3. Flex-grow:     It controls the amount of available space an element should take up. Accepts a unit-less number value. It accepts an integer number like 1,2,3. The bigger the number the more times an element occupies the space. When we start to reduce the screen size it all comes to a point where all the elements occupy equal space and when we increase the size it again becomes the bigger element depending upon the flex-grow property applied unless flex-wrap is turned on then after minimum threshold it will move to the next line.
# Property Declaration
1 flex-grow: 1 The element occupies double the size of the other elements with in the container
2 flex-grow: 2 The element occupies 3 times the size of the default size element with in the container
4. Flex-shrink:     If items are larger than container, they shrink according to flex-shrink, When we apply the property flex-shrink property we are just telling the element how much it should shrink relative to other elements
# Property Declaration
1 flex-shrink: 1; When we reduce the screen size, it will shrink the element with double the rate on which flex-shrink:1 is applied.
2 flex-shrink: 0; When we reduce the screen size, it will not shrink the element's size at all and rest of the elements will shrink as per the screen size.
5. Order:     It specifies the sequence or order which is used to lay out items in their flex container. They are useful when we have more than one items in a container as we can change the sequence using the property Order. We can assign a value in negative and positive both, the default value is "0".
# Property Declaration
1 order: 0 Its is the default value which lets an item stay as per the regular sequence
2 order: -1 It puts an item ahead of the other items with in a container, the higher the negative value will be the more further an item will be placed
3 order: 1 It puts an item after the default item place in a container, the higher the positive value will be, the later an item will be placed with in a container.
